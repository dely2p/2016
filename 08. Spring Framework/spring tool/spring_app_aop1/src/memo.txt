AOP(Aspect Oriented Programming)
: 관점 지향 프로그램/코드의 삽입으로 사용됨

여러 객체에 공통으로 적용할 수 있는 기능을 분리 함으로서 재사용을 높여주는 프로그래밍 기법

은행?

 핵십기능 : 입금, 출금, 송금 ..
 
 공통기능 : 로그인, 권한 검사, 감사 기록
 
 OOP : C++, Java, C#
 
 AOP : AspectJ XXXX
 
 AOP를 적용하는 방법
 1. 컴파일 시점에 코드를 공통 기능에 추가하는 방법
 2. 클래스 로딩 시 바이트 코드에 공통 기능을 추가하는 방법
 3. 런타임시에 프록시 객체를 생성해서 공통기능을 추가하는 방법(spring은 이것만 지원함)
